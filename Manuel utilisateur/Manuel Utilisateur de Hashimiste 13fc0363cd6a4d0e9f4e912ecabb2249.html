Introduction :

Bienvenue dans Hashimiste, un jeu de puzzle stimulant où vous devez connecter toutes les îles par des ponts selon certaines règles. Ce manuel vous aidera à naviguer dans le jeu et à découvrir des stratégies pour résoudre les puzzles.

Configuration Requise :

Windows :

  • Version : Avoir une version de Windows supérieure ou égale à Windows XP.
  • Java : Avoir Java 8.

Linux/Mac :

Pour Mac :

  • Version : Avoir une version de macOS 10.7.3 (Lion) ou ultérieure.
  • Java : Avoir Java 8.

Pour Linux :

  • Noyau Linux : Version 2.6.32 ou ultérieure.
  • Bibliothèques : glibc 2.12 ou supérieur.
  • Java : Avoir Java 8.

Installation du jeu :

Lors de l’installation du jeu, il sera nécessaire d’avoir java 8 d’installé sur sa machine : https://www.java.com/fr/download/

Si java 8 est bien installé sur la machine, il reste plus qu’à télécharger le jeu :

https://drive.google.com/file/d/1QgdECVEMzMarRNGMTbvn2mMZUYFxkEro/view?usp=drive_link

Après avoir installé le jeu, vous devez simplement double-cliquer sur le .jar.

Démarrage du jeu :

Après avoir créé ou sélectionné votre profil, pour démarrer une partie, vous devez sélectionner un mode de jeu (vous pouvez aller voir les techniques mises à disposition avant). Ensuite, vous devez, selon les différents modes de jeux, soit charger une partie déjà commencée ou bien sélectionner la difficulté de la grille sur laquelle vous souhaitez jouer. Par exemple, en mode aventure, les grilles sont déjà générées, donc vous ne pouvez pas les modifier.

Puis une grille va apparaître et vous devez, à l’aide des commandes, compléter la grille tout en respectant les règles du jeu expliquées ci-dessous.

Règles du jeu :

Dans le jeu, chaque île doit être connectée aux autres par un certain nombre de ponts, qui est indiqué sur l'île elle-même par un chiffre situé entre 1 et 8. Les ponts ne peuvent traverser ni les îles, ni les autres ponts, et tous les ponts doivent former un chemin continu entre toutes les îles.

Commandes de jeu :

Afin de jouer au jeu, vous devez connaître que très peu de commandes. Ces commandes sont la pose d’un ou de deux ponts, la suppression des ponts et la visualisation des ponts possibles.

Pour la pose des ponts, en survolant l’espace entre deux îles, un pont semi-transparent apparaît. Avec un clic gauche dessus, vous créez ce pont. En cliquant une seconde fois dessus, vous créez un deuxième pont au même endroit.

Pour supprimer les ponts, il suffit d’interagir avec un double pont pour le supprimer à l’aide d’un clic gauche.

Concernant la visualisation des ponts, en survolant une île, vous voyez les ponts pouvant être créés.

Navigation dans le jeu :

Création/Chargement du profil :

Dans cette section, vous pouvez créer un nouveau profil ou charger un profil existant. Votre progression dans le jeu sera enregistrée sous le profil que vous utilisez.

Dans le premier cas, vous possédez déjà un profil, vous avez juste à cliquer dessus pour reprendre votre compte.
Dans le second cas, vous ne possédez pas de profil ou vous souhaitez en créer un nouveau, vous devez donc cliquer sur le bouton “+” et saisir un nom de profil dans la zone de texte indiquée.

Après avoir saisi votre nom de profil, vous devez cliquer sur le bouton “OK”. Vous n’avez plus qu’à suivre le premier cas en cliquant sur votre profil pour pouvoir reprendre votre compte.

Aventure :

Dans le mode Aventure, vous suivez une série de niveaux avec une difficulté croissante. Chaque niveau présente un nouveau défi à relever.

Tout d’abord, vous devez cliquer sur le bouton “Aventure”.

Puis cliquer sur le niveau disponible selon votre avancement.

Mode Libre :

Dans le mode Libre, vous pouvez choisir parmi une variété de puzzles à résoudre à votre rythme. Il n'y a pas de restrictions de temps ou de niveau dans ce mode.

Tout d’abord, vous devez cliquer sur le bouton “Mode libre”.


Vous avez le choix entre trois modes de difficultés, Facile, Moyen et Difficile.

En passant la souris sur une carte, cela donne des informations comme la taille, le nombre de parties jouer, le meilleur temps etc.

Vous avez plus qu’à cliquer sur l’une des cartes pour lancer une partie.

Technique :

Dans la section Technique, vous pouvez voir différentes techniques pour avancer dans le jeu.

Vous avez juste à cliquer sur l’une des techniques, allant de la plus simple à la plus avancée. Nous conseillons au débutant d’aller voir seulement les premières techniques lors de leurs premières parties.

Paramètres :

Dans la section Paramètres, vous pouvez changer le thème du jeu selon vos préférences.

Après avoir choisi le thème souhaitez, il faut cliquer sur le bouton “Sauvegarder” en bas à droite de la fenêtre de jeu. ( Actuellement, le thème choisit est le thème “Candy” )

En partie :

Ajout et suppression de pont :

Lors de vos parties, vous devez poser ou supprimer des ponts pour pouvoir résoudre le niveau.

En passant la souris entre deux iles, un pont en surbrillance indique que vous pouvez poser un pont en faisant clic gauche.

Vous pouvez rajouter un deuxième pont en faisant de nouveau un clic gauche.

Puis, si vous souhaitez enlever un pont, vous avez juste à faire un clic droit.

Progrès et sauvegarde :

Lors d’une partie, vous pouvez choisir de faire une sauvegarde pour reprendre votre partie actuelle plus tard.

Vous devez cliquer sur le bouton “Sauvegarder”, une fenêtre va s’ouvrir.

Vous devez maintenant rentrer un nom de sauvegarde puis cliquer sur “OK”.

Vous pouvez aussi choisir de faire une sauvegarde rapide dites “Checkpoint”.

Vous avez juste à cliquer sur le bouton “Checkpoint”.

Par la suite, si vous souhaitez reprendre votre Sauvegarde ou votre checkpoint, vous devez cliquer sur le bouton Charger.

Ensuite, vous aurez une fenêtre vous demandant quelle sauvegarde choisir.

Choisissez la sauvegarde que vous souhaitez reprendre puis vous cliquez sur le bouton “OK”.

Aide :

Pour avoir de l’aide, vous devez cliquer sur le bouton “Aide”.

Puis une boite de dialogue va vous indiquer une aide.

Et si l’aide ne suffit pas, on peut avoir plus de précision.

Vérification :

Si vous souhaitez une vérification de la bonne avancement de votre niveau, vous pouvez cliquer sur le bouton “Vérifier”.

Puis une boite de dialogue va vous indiquer si votre schéma est correct ou non

Retour :

Vous pouvez effectuer un Retour arrière en jeu qui permet de revenir une action en arrière. Pour faire cela, vous avez juste à cliquer sur le bouton “Retour”.

FAQ :

Q : Le bouton Tutoriel et Multijoueur ne marche pas ?

R : Oui, le tutoriel et le multijoueur ne sont pas encore disponibles, ils le seront dans le futur.

Q : Puis-je créer mes propres cartes de Hashi et y jouer ?

R : Un éditeur de carte existe pour les développeurs, il sera bientôt implémenté au jeu !

Q : Mon ami souhaite jouer sur ma machine sans pour autant être sur ma partie, existe-t-il une solution ?

R : En effet ! Un menu profil permet de conserver une progression différente pour chaque utilisateur.

Q : Les couleurs ne sont pas à mon gout, puis-je les changer ?

R : Tout à fait ! Dans le menu Paramètres, divers thèmes sont disponibles.

Q : Je n’ai pas envie d’utiliser d’aide pendant une grille, est-ce possible de sauvegarder mon avancée avant de tester des manipulations ?

R : Effectivement, le bouton Checkpoint permet une sauvegarde rapide pour y revenir sans danger.
